# 🍽️ BistroFlow – Smart Restaurant Reservations

**Smart Seating, Flawless Reservations**  
BistroFlow is a modern web application that lets diners explore live table availabilities, select preferred layouts, and lock in reservations instantly.

---

## 🚀 Features
- 🏠 Home Page – Hero section with quick actions (Book Table, Login, Register)
- 📖 About Page – BistroFlow’s mission and smart seating technology
- 📞 Contact Page – Hotline, email support, and optional contact form
- 🔑 Register/Login – Secure authentication with account type selection
- 📅 Book Table – Interactive floor plan matrix with real-time availability
- 📂 My Bookings – View and track confirmed reservations
- ⏳ Waitlist – Join live queue when tables are full

---

## 🛠️ Tech Stack
- **Frontend:** React + Vite  
- **Backend:** Node.js, Express  
- **Database:** MongoDB
